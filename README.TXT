本项目简介
	提供GIT中国完成文章发帖关键字过滤功能、同时免费提供该服务
本项目技术架构设计
	采用主流的REST面向服务面向组件的架构设计思想下采用SpringMVC的REST技术实现规范的实现。
	SpringMVC+Maven+Git(版本控制)
